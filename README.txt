PaySimple for WooCommerce Payment Gateway authorizes credit card payments and processes them securely with your merchant account.

== Description ==

WooCommerce PaySimple Payment Gateway Integration enables you to accept credit card payments securely with your paysimple merchant account.

This plugin is for those users are using PaySimple as payment gateway and want to integrate PaySimple with WooCommerce shop. 

= Plugin Functionality: =

* Easy to install and configure.
* Compatible with WordPress/WooCommerce plugins
* You don’t need any extra plugins or scripts to process the Transaction
* Accepts all major credit cards
* Refunds functionality available.
* Tested this plugin on the PaySimple sandbox (test) servers to ensure your customers don't have problems paying you.
* Our plugin handles warnings/errors correctly thrown in response so that user do not have to worry about dealing with those types of situations.
* Log write functionality available for transactions.

We always welcomes user suggestions. Let us know what you think about this plugin you liked or may have disliked. Users feedback is important for us to improve more our plugin.
